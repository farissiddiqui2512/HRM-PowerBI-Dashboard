# HRM Dashboard - Power BI

This project is an HR Management Dashboard created in Power BI.

## Features
- Total Employees
- Age Distribution (Bins)
- Salary Analysis
- Gender & Job Title breakdown
- Leave Balance overview

## Tools Used
- Power BI Desktop
- Excel (Data Source)
- Power Query
- DAX

## How to Use
1. Download the `.pbix` file
2. Open it in Power BI Desktop
